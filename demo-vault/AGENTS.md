# Demo Vault

This is the demo vault for Ax Visual Lab — a minimal Obsidian-compatible vault that gives the visual lab something to browse when run in demo mode.

## For future agents

This vault exists solely to demonstrate the vault tools (read, search, list) in the visual lab. It contains sample notes that are safe to read, modify, or delete.

When you connect a real Obsidian vault, set `OBSIDIAN_VAULT_PATH` in `.env` and restart.
